#  Exercício: Calculadora Aritmética Dinâmica

Este exercício foi desenvolvido para praticar e fixar o uso de **estruturas condicionais encadeadas** (`if-else`) na linguagem Java. O objetivo principal é construir um fluxo de controle dinâmico onde o programa decide qual operação matemática executar com base na escolha informada pelo usuário no console.

##  Funcionamento do Programa

1. O programa solicita que o usuário insira dois números inteiros.
2. Em seguida, exibe um menu numérico com quatro opções de operações aritméticas:
   * `1` - Somar
   * `2` - Subtrair
   * `3` - Dividir
   * `4` - Multiplicar
3. O usuário digita o número correspondente à operação desejada.
4. O programa avalia a escolha, realiza o cálculo específico e exibe o resultado final.
5. Caso o usuário digite qualquer número fora do intervalo de 1 a 4, o sistema dispara uma mensagem de aviso: `"Numero inválido."`.

---

##  Conceitos Praticados

* **Entrada de Dados (`Scanner`):** Utilização da classe `java.util.Scanner` para capturar dados primitivos inseridos via teclado (`nextInt`).
* **Operadores Aritméticos:** Aplicação das operações fundamentais do Java: adição (`+`), subtração (`-`), divisão (`/`) e multiplicação (`*`).
* **Estrutura de Decisão (`if / else if / else`):** Desvio de fluxo condicional para garantir que apenas o bloco de código da operação escolhida seja executado.
* **Saída de Dados (`System.out.println`):** Impressão de mensagens informativas e concatenação de strings com variáveis de resultado no console.
